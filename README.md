Deployment Notes
----------------

Ensure environment is setup with the following.
```
export MONGO_NODE_DRIVER_HOST='sfwme.com'
export MONGO_NODE_DRIVER_PORT=27017
export NODE_ENV='production'
export BASE_URL='sfwme.com'
```
